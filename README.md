# Spotify Big Data Analysis

## Project Overview

This project analyzes a Spotify dataset containing more than 10,000 songs using PySpark and Spark SQL. The objective is to perform data cleaning, preprocessing, exploratory data analysis (EDA), and data visualization to gain insights into song characteristics and popularity trends.

## Dataset

* Dataset: Spotify Songs Dataset
* Total Records: 10,000+ songs
* Format: CSV

## Tools & Technologies

* Python
* PySpark
* Spark SQL
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Project Objectives

* Clean and preprocess raw Spotify data.
* Explore song characteristics and popularity metrics.
* Perform data analysis using Spark SQL queries.
* Generate visualizations to support data interpretation.
* Extract meaningful insights from large-scale music data.

## Analysis Process

1. Data Collection
2. Data Cleaning
3. Data Preprocessing
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Insight Generation

## Key Tasks

* Handling missing values
* Data transformation using PySpark
* Querying data using Spark SQL
* Feature exploration
* Statistical analysis
* Data visualization
